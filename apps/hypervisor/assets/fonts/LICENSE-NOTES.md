# Vendored font assets

- `source-sans-pro-400.woff2`, `source-sans-pro-600.woff2`, `source-sans-pro-700.woff2` —
  **Source Sans Pro** (Adobe / Paul D. Hunt), licensed under the **SIL Open Font License 1.1**
  (https://scripts.sil.org/OFL). The OFL permits bundling and redistribution. These files serve the
  IOI-owned `/__ioi/*` surfaces via the `/__ioi/fonts/<name>.woff2` route so pixel-certified ports render
  with the same type metrics as the reference captures.

No proprietary fonts are vendored. Blueprint icons are reproduced as inline SVG paths
(@blueprintjs/icons, **Apache-2.0**) directly in the surface markup — no icon font is bundled.
