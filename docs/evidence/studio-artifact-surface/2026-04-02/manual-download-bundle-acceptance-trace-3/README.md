# A downloadable artifact bundle with a CSV and README

This bundle contains the requested downloadable files and a short explanation of how to use them.

## Files
- `README.md`: Bundle overview, file mapping, and CSV column notes.
- `data.csv`: Structured CSV export for the requested bundle.

## CSV columns
- `record`: Label for the bundle record described in the export.
- `detail`: Request-grounded detail for that record.

## Notes
- The bundle should include a non-empty README.md that explains the bundle contents and how each file maps to the request.
- The CSV export should have a header row plus at least two data rows with request-grounded values.
