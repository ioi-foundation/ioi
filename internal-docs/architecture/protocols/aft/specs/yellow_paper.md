# Asymptote Fault Tolerance (AFT) Yellow Paper

The internal yellow paper source is maintained in
[`yellow_paper.tex`](./yellow_paper.tex).

Build it from this internal location with
`./.github/scripts/build_aft_yellow_paper.sh`, which writes the generated PDF
and TeX build artifacts into
[`internal-docs/formal/aft/generated/specs`](../../../../formal/aft/generated/specs/)
rather than the repo root.

This markdown file is retained only as a lightweight compatibility stub for
tools or links that still expect a `.md` sibling at this path.
